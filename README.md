# Installing

Stable library from PyPi:

* Just run `python3 -m pip install pimoroni-sgp30`

Latest/development library from GitHub:

* `git clone https://github.com/pimoroni/sgp30-python`
* `cd sgp30-python`
* `./install.sh --unstable`
